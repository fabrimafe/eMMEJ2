# This folder contains plotting scripts of genomic data


```
├── DGRP_exon_non_genic_indel_len_dist.py -> generates barplots of 
|       indel length density in exons and non-genic regions.
├── EMmej_results_plotting_general.py -> generates boxplots of a given two
|       categories, as well as barplots with estimations per indel length.
├── get_mmej_cand_len_distribution.py -> generates the microhomology length distribution
├── hist_mod_plots.py -> generates hostone modifications boxplots.
├── README.md
└── repeat_ann_plots.py -> generates boxplot with 
            estimations of different repeated regions
```
