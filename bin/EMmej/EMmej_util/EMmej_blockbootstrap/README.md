# This folder contains all scripts that involve in batch/blockbootstraping using EMmej

## One can operate the whole thing as follow: 
    - run batch_data_and_submit_EMmej_to_cluster.py (divide data to blocks/batchs of a given size, then run EMmej)<br>
    - run EMmej_blockbootstrap.py (performing blockbootsrtaping) <br>
    - run EMmej_blockbootstrap_summarizer.py (summerizrs the blockbootsrtaping results)<br>

the script: whole_genome_batching_by_blocks.py is used within batch_data_and_submit_EMmej_to_cluster.py.
