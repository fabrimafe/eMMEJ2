# emMEJ2
This is an update over emMEJ (which was built with Guy Tadmor) to replace 2nd order Markov chain approach is my new way to document my master's project
emMEJ can be found in https://github.com/GuyTadmor-28/guy_master_project or in local WEXAC folder /home/labs/alevy/fabrizio/github/guymasterproject_final/guy2023

## Directory description:
```
.
├── data - Folder that contains data per plant per publication
├── docs - Scripts and modules documentations
├── environment.yml - Old venv
├── guy_mmej_env.yml - Current venv
├── README.md 
├── results - Folder contains all results per plant per publication
├── scripts - Folder contains all scripts
├── src - Folder contains all callable modules
└── tests - Folder contains all test files
```
## Venv
To recover the venv for this project, use:
module load miniconda
conda create --name recoveredenv --file guy_mmej_env.yml
